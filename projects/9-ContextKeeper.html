ContextKeeper

A community-driven fork of NVIDIA's G-Assist project, expanding the plugin ecosystem for AI-powered PC management.

Overview

ContextKeeper enables RTX GPU users to manage PC settings and applications through voice or text commands, with a strong focus on extensibility and community contributions.

Key Features

  • Voice & Text Control: Natural language interaction for PC management
  • Extensible Plugin Architecture: Build custom functionality with Python or C++
  • Rich Plugin Ecosystem: Pre-built integrations for popular services
  • Python API Bindings: Easy integration and development
  • GPU-Accelerated: Leverages NVIDIA RTX capabilities

Example Plugins

  • Twitch Integration: Stream management and chat interaction
  • Spotify Control: Music playback and playlist management
  • Peripheral Lighting: Logitech, Corsair, and Nanoleaf support
  • Custom Automations: Build your own integrations

Technical Stack

  • Primary Language: Python (81.2%)
  • Performance Components: C++ (12.7%)
  • Plugin Invocation: Zero-shot function calling or direct specification

Development

The project emphasizes community contributions and provides comprehensive documentation for plugin development. Whether you're building simple automations or complex integrations, ContextKeeper provides the framework for extensible AI assistance.

Perfect for power users and developers looking to create custom AI-powered workflows!
