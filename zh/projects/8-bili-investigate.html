Bili-Investigate

A smart web application for tracking and managing Bilibili content creators' video catalogs.

Problem Statement

Obtaining comprehensive video lists from Bilibili content creators is challenging, particularly for educational series and long-running content. This tool addresses that need with an intuitive interface.

Features

  • Smart Incremental Updates: Intelligent fetching to avoid redundant data collection
  • Creator Management: Easy interface for following and managing content creators
  • Complete Video Catalogs: Browse full video histories of your favorite creators
  • Advanced Search: Find specific videos across all tracked creators
  • Flexible Sorting: Multiple sorting options for better organization
  • Local Storage: JSON and CSV formats for data persistence

Technical Details

  • Language: Python (71.2% of codebase)
  • Framework: Streamlit for the web interface
  • Deployment: Docker support for easy setup
  • Platforms: macOS and Linux compatible

Use Cases

Perfect for: - Researchers tracking educational content - Fans following multiple creators - Content curators building collections - Anyone needing comprehensive Bilibili video catalogs
