# BTMR (Beautiful Text Mining Reader)

An intelligent system designed to help researchers and students quickly digest academic papers with AI-powered analysis.

## Features

- **Multi-source Processing**: Support for ArXiv URLs, PDFs, and web links
- **AI-Powered Summarization**: Structured summaries using advanced LLMs
- **Multiple Output Formats**: HTML and PDF with embedded figures
- **Multilingual Support**: English and Chinese
- **Modern Tech Stack**: FastAPI backend with React 18 frontend
- **Real-time Updates**: WebSocket support for progress tracking
- **Easy Deployment**: Docker support with comprehensive API documentation

## Technical Stack

- **Backend**: Python with FastAPI
- **Frontend**: TypeScript/React 18
- **Database**: SQLite for metadata management
- **API**: RESTful with Swagger UI and ReDoc interfaces

## Key Capabilities

- Drag-and-drop file upload interface
- Intelligent paper content extraction
- Automated summarization and analysis
- Responsive web interface
- Comprehensive API for integration

Perfect for researchers who need to process large volumes of academic papers efficiently!
